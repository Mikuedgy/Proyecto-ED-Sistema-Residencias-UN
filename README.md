++DESCRIPCION

En la Universidad Nacional se prestaba el servicio de residencias universitarias destinadas al apoyo y bienestar de los estudiantes más vulnerables. Las residencias son limitadas y exigen una alta demanda por lo cual es menester desarrollar una solución de gestión eficiente y justa.

Dentro de la universidad se manejan puntajes PBM, un puntaje (socioeconómico) y cada estudiante está asociado a uno. Entre menor sea el puntaje del estudiante, mayor es su prioridad para tener un cupo en las residencias.

El sistema a implementar debe tener la capacidad de registrar, consultar, eliminar y modificar cada dato particular de los usuarios al igual que asignar disponibilidad de cupos con base al criterio del puntaje PBM. Mantener un correcto orden y eficiencia a la hora de manejar y modificar esta información al igual que controlar un buen tamaño de datos es el principal reto a enfrentar.
